# dental-clinic-dentist-api-flask

The code is generated by:
```
swagger_py_codegen -s swagger.yaml dentistAPI --ui --spec
```

## Get started using Docker container

Build the Docker image:
```
docker build -t dentist-api .
```

Run the Docker container:
```
docker run -p 8083:5001 -t dentist-api __init__.py
```

The GUI of the API is at:
```
http://0.0.0.0:8083/static/swagger-ui/index.html#/
```
(Note: the exposed container port 5001 is mapped to the local machine port 8083)

Use the API from:
```
http://0.0.0.0:8083/v1
http://0.0.0.0:8083/v1/dentist
http://0.0.0.0:8083/v1/dentist?name=Dr+Green
http://0.0.0.0:8083/v1/dentist/2
```
